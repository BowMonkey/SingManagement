题目3  歌手比赛成绩管理系统  
该系统的目标是对一次歌手比赛的成绩进行管理，减少人工计分造成的失误。
基本要求
1、输入每个选手的数据包括编号、姓名、十个评委的成绩，根据输入计算出总成绩和平均成绩（去掉最高分，去掉最低分）。
2、显示主菜单如下：
1）输入选手数据 
2）评委打分 
3）成绩排序（按平均分）
4）数据查询  
5）追加学生数据 
6）写入数据文件
7）退出系统。
